cdp_langchain.tools.cdp_tool module¶

Tool allows agents to interact with the cdp-sdk library and control an MPC Wallet onchain.

To use this tool, you must first set as environment variables:

CDP_API_KEY_NAME CDP_API_KEY_PRIVATE_KEY NETWORK_ID

response_format: Literal['content', 'content_and_artifact']¶

The tool response format. Defaults to ‘content’.

If “content” then the output of the tool is interpreted as the contents of a ToolMessage. If “content_and_artifact” then the output is expected to be a two-tuple corresponding to the (content, artifact) of a ToolMessage.

return_direct: bool¶

Whether to return the tool’s output directly.

Setting this to True means that after the tool is called, the AgentExecutor will stop looping.
